Conexión con APIs

Conexión Gmail
No conectado

Conecta tu cuenta de Gmail para gestionar tus correos electrónicos directamente desde la plataforma.

Para conectar con Gmail API:
  1. Ve a la Consola de Google Cloud
  2. Crea un nuevo proyecto o selecciona uno existente
  3. Habilita la API de Gmail para tu proyecto
  4. Crea credenciales de tipo OAuth 2.0 Client ID
  5. Configura la pantalla de consentimiento de OAuth
  6. Añade tu dominio y URLs de redireccionamiento
  7. Copia el Client ID y API Key en los campos anteriores
Conexión OpenAI
No conectado

Conecta con OpenAI para generar respuestas automáticas inteligentes y asistencia en la gestión de correos.

Para obtener tu API Key de OpenAI:
  1. Ve a la plataforma de OpenAI
  2. Inicia sesión o crea una cuenta nueva
  3. Accede a la sección "API Keys" en tu dashboard
  4. Haz clic en "Create new secret key"
  5. Copia la clave generada y pégala en el campo anterior
  6. Asegúrate de tener créditos suficientes en tu cuenta
